What is F1 Tracker?

F1 Tracker is a full stack web application for exploring F1 race telemetry. Users can select any Grand Prix session from 2023 to 2025, choose two drivers, and compare their laps side by side viewing speed, throttle, brake, RPM, and gear traces as synchronized Chart.js line graphs, plus S1 / S2 / S3 sector times.

The home page shows the latest standings for constructor and driver championship. As well as a live countdown to the next race

How to Use Telemetry

  • Select a Season year from the dropdown
  • Select a Grand Prix from the list that appears
  • Select the session type (Race, Qualifying, Practice)
  • Click Load Session to pull driver and lap data
  • Click a lap row in the Driver A table, then one in Driver B
  • Click Compare Telemetry — charts and sector pills appear

Node / Express
Backend REST API serving JSON to the frontend.
